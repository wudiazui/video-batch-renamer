视频批量重命名工具 - 源码交付说明

本包只包含源码，不包含 EXE。

目录结构：
  src\              程序源码（main.py 为入口）
  tests\            单元测试（33 个）
  README.md         开发者说明
  使用手册.md       面向最终用户的图文操作手册（含 FAQ、错误解决表）
  更新日志.md       版本变更记录
  run.bat           一键启动脚本（双击即可）
  build_exe.ps1     打包 EXE 脚本
  requirements.txt  依赖说明（运行无需第三方库）

运行源码：
1. 安装 Python 3.10 或更高版本（安装时勾选 Add Python to PATH）。
2. 在当前目录双击 run.bat，或运行：python src\main.py

运行测试：
  python -B -m unittest discover -s tests

如以后需要自行打包 EXE：
1. 先安装 PyInstaller：pip install pyinstaller
2. 再运行：powershell -ExecutionPolicy Bypass -File .\build_exe.ps1
   打包结果在 release\视频批量重命名工具.exe
