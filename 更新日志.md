# 更新日志

## v1.2.2 · 2026-06-18

### 修复
- **撤销不再被「空操作」卡住**：识别集数模式下，若某文件改名后与原名相同（如本来就叫 `第1集.mp4`），之前点「撤销最近一次」会误报「目标已存在」、导致整批无法撤销。现在正确跳过这类空操作、撤销其余文件。（由可视化端到端测试发现）

## v1.2.1 · 2026-06-18

### 识别集数增强
- 新增「松散兜底」：文件名带数字但没有「集/话」标记时也能识别，例如 `超清-6`、`EP6`、`第6话`、`S01E06`、`1080P-6` 都会认成第 6 集。
- 自动剔除清晰度/编码/帧率/版本等噪声（`1080P`、`720P`、`2k`、`4K`、`x264`、`1920x1080`、`60帧`、`v2` 等），避免把它们误当成集数；纯清晰度名（如 `1080P`、`2k`）与四位年份（如 `2024`）仍正确地不识别。
- 取「剔除噪声后最后一个 1~3 位独立数字」为集数。

### 手动设置集数
- 预览清单里**右键任意一行可手动指定第几集**：自动识别失败的（红色行）填上集数即可改名；识别对的也能手动覆盖。改完即时重算冲突与可执行状态。

## v1.1.1 · 2026-06-17

代码审查后的补充修复，均带回归测试，单元测试增至 36 个。

### 修复
- **进度条不再滞后**：批量执行时，进度消息在每次刷新时一次性排空，只用最新一条更新进度条；几百个文件也能实时反馈、完成弹窗及时弹出（此前每 100ms 只处理一条，会严重滞后）。
- **执行/撤销期间拦截快捷键**：新增“忙碌”状态判断。执行进行中再按 `Ctrl+Enter` / `Ctrl+Z` 不会再启动第二个执行线程或插入撤销，杜绝并发改名竞态。
- **执行中改设置不再重置进度条**。
- **跨卷移动更稳**：`move_no_overwrite` 改用 `shutil.move`——同卷仍走原子 rename，遇到跨盘符 / 网络映射（含目录联接 junction）时自动回退“复制 + 删除”，不再整批失败；仍保证不覆盖已存在文件。

### 体验
- **后台统计视频数**：添加超大文件夹时不再卡界面，视频数量异步回填到列表。
- **双击预览行定位文件**：在预览清单里双击任意一行，可在资源管理器中选中对应文件。

### 整洁
- 统一 `READY_STATUS` 常量、删除未使用的死代码、为模式/模板不匹配给出明确报错。
- 重命名日志 CSV 的 `timestamp` 列现在会写入执行时间（此前恒为空）。
- 《使用手册.md》补充界面分区图与四步流程图（`docs/images/*.svg`），成为图文版。

## v1.1 · 2026-06-17

本次围绕「代码检查 → 优化 → 交互改进 → 配套文档」做了一轮整体打磨。

### 交互体验
- **统一文件夹列表**：去掉了原来「选择文件夹」与「队列」并存、容易混淆的双轨设计，现在所有操作都基于一个列表。列表会显示每个文件夹的视频数量和总计。
- **跨文件夹连续编号**（新功能）：连续编号模式下可勾选，多个文件夹接续编号（A→1,2,3，B→4,5）；文件仍各自留在原文件夹，不会合并。
- **改设置即失效旧预览**：生成预览后只要改了任意命名设置，执行按钮立即禁用并提示重新预览，避免按旧规则误执行。
- **撤销支持多文件夹**：「撤销最近一次」会在列表全部文件夹中找时间最新的日志，而不再只看第一个文件夹。
- **模式记忆模板**：连续编号与识别集数各自记住上次用的模板，切换模式不再互相覆盖。
- **友好错误提示**：预览出现红色项时，底部用一句话说明该怎么解决。
- **执行期间禁用所有按钮与列表**，防止误触。
- **悬停说明气泡**：鼠标停在命名设置区的控件（连续编号 / 识别集数 / 起始数字 / 跨文件夹连续编号 / 剧名 / 模板 / 保持扩展名大小写）上，会弹出一句话说明。
- **细节**：所有弹窗归属主窗口（不再可能藏到窗口后面）；记住窗口大小与位置；新增快捷键（Ctrl+O/P/Enter/Z、Delete）；列表支持右键菜单与双击在资源管理器打开；预览表头显示排序箭头；空白时显示操作指引。

### 正确性 / 健壮性
- **拦截 Windows 系统保留名**：剧名或模板生成 `CON/PRN/NUL/COM1…/LPT1…` 等名字时会在预览中标红，避免执行失败。

### 文档与交付
- 新增面向最终用户的《使用手册.md》（含命名模板对照、错误解决表、FAQ）。
- 修正 README.md 中过时的硬编码路径，补充新功能与快捷键说明。
- 新增一键启动脚本 `run.bat`。
- 加固 `build_exe.ps1`：打包前检查 Python 与 PyInstaller 是否就绪。
- 测试从 22 个增加到 30 个，覆盖跨文件夹连续编号、保留名拦截、中文数字（十一/二十/九十九/一百零八）及关键交互逻辑。

经核查，之前怀疑的几个「严重 Bug」实际并不成立：中文集数「十一/二十/九十九」原本就能正确识别；撤销方向、跨磁盘移动在本工具的使用场景下也没有问题（视频只在同一根目录树内移动）。因此本次未做无谓改动。
